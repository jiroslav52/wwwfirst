Советы по уходу

Наши растения неприхотливы, но любят внимание. Ниже — основные правила для каждой группы.

Фикусы

  • Свет: яркий рассеянный, прямые лучи могут обжечь листья
  • Полив: 1–2 раза в неделю, когда верхний слой грунта подсох
  • Температура: 18–24 °C, без сквозняков
  • Опрыскивание: 2–3 раза в неделю мягкой водой
  • Подкормка: раз в 2 недели весной и летом

Суккуленты

  • Свет: максимум солнца, лучше на южном окне
  • Полив: редкий — раз в 1–2 недели, грунт должен полностью просыхать
  • Температура: летом 22–28 °C, зимой 10–15 °C для покоя
  • Опрыскивание: не нужно, не любят влагу на листьях
  • Грунт: специальный для кактусов и суккулентов с дренажём

Монстеры

  • Свет: яркий рассеянный, переносит лёгкую полутень
  • Полив: 1 раз в неделю, любит стабильно влажный грунт
  • Температура: 20–25 °C круглый год
  • Опрыскивание: ежедневно или протирайте листья влажной тканью
  • Опора: установите кокосовый столбик для воздушных корней

Пальмы

  • Свет: яркий рассеянный, без прямого солнца
  • Полив: 2 раза в неделю летом, 1 раз зимой
  • Температура: 18–22 °C, не ниже 16 °C
  • Опрыскивание: ежедневно для сохранения свежести листьев
  • Пересадка: раз в 2–3 года, не любят тревогу корней

Цветущие

  • Свет: яркий рассеянный для обильного цветения
  • Полив: умеренный, грунт всегда слегка влажный
  • Температура: 20–25 °C во время цветения
  • Подкормка: специальное удобрение для цветущих раз в 2 недели
  • Удаляйте отцветшие бутоны для продления цветения
